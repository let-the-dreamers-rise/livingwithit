# LIVING WITH IT — Build Plan

## Context

A Figma Makeathon submission ($50k first-prize goal). The build is a mobile-first
web app, **LIVING WITH IT** — a _bounded, nine-week reflective instrument_ for
people who already made an irreversible decision and cannot stop doubting it.

The differentiator (the "moat" judges reward): unlike ChatGPT, this app **enforces
its constraints in code** via input _gates_ that refuse to re-litigate a locked
decision. It is a ritual instrument, not a chatbot. The full product spec lives at
`src/imports/pasted_text/living-with-it.md` and is the source of truth for copy,
gates, phases, and design tokens.

**Decision made during planning:** weekly reflections are generated by **Claude via
a Supabase edge function**, used as a _pure ephemeral relay_ — no user data is
stored server-side, preserving the app's "no cloud profiling / session-scoped"
philosophy. Gates run client-side _before_ any call. A template fallback ensures a
live demo never breaks offline.

## Architecture

**State machine (4 phases):** `VOID → COMMITMENT (Week 1) → CHAMBER (Weeks 2–9) → TERMINAL`.
Implement with `useReducer` so illegal transitions are impossible. The decision is
immutable after Week 1 (no edit UI exists in the DOM, not merely disabled).

**Data flow per check-in:**

1. User submits question → client-side gates validate (RE-LITIGATION, DEPTH, ANTI-COMPARE).
2. On reject → `GatePanel` shows label/reason/hint, no network call, field kept for rewrite.
3. On accept → call Supabase edge function `generate-reflection` → Claude → render
   reflection with 800ms fade. On any error → template fallback, silently.

## Supabase (first implementation step)

Use the `make:supabase` skill to connect the project and create the edge function.

- **Edge function `generate-reflection`**: receives `{ decision, weekNumber, weekLabel, question }`,
  calls the Anthropic API (`claude-opus-4-8` or `claude-sonnet-4-6`) with a strict
  system prompt enforcing: first-person future-self voice; tone arc by week
  (1–2 disorientation/grief, 3–5 emerging costs, 6–9 integration); **never** advice,
  prediction, or validation/invalidation of the decision; always ends with the
  muted disclaimer line. Returns `{ reflection }`. **Stores nothing.**
- Secret `ANTHROPIC_API_KEY` via the Supabase secret card.
- The archive remains **localStorage only**.

## Files

- `src/app/App.tsx` — root: owns the `useReducer` state machine, routes phases, handles
  the archive (load/save to localStorage), calls the edge function.
- `src/app/lib/gates.ts` — pure functions: `commitmentGate`, `reLitigationGate`,
  `depthGate`, `antiCompareGate`. Each returns `{ ok } | { ok:false, label, reason, hint }`.
  Patterns copied verbatim from the spec. (Unit-testable, the product moat.)
- `src/app/lib/reflection.ts` — `fetchReflection()` (calls edge fn) + `templateReflection()`
  fallback engine parameterized by decision keywords / week / question.
- `src/app/lib/weeks.ts` — week label map (Week 1 "One week in" … Week 9 "A lifetime in").
- `src/app/lib/archive.ts` — localStorage read/write of completed arcs.
- `src/app/lib/export.ts` — generate + download the `.md` Living Record.
- `src/app/components/`:
  - `VoidState.tsx`, `CommitmentRitual.tsx`, `WeeklyChamber.tsx`, `TerminalState.tsx`
  - `GatePanel.tsx`, `LockedDecisionBanner.tsx`, `WeekCounter.tsx`, `WeekLabelBadge.tsx`,
    `ReflectionBlock.tsx` (800ms fade), `LivingArchive.tsx` (list + read-only tombstone),
    `DemoNote.tsx`.
- `src/styles/fonts.css` — `@import` Playfair Display (headlines/reflections) + Inter (body)
  at top of file only.
- `src/styles/theme.css` — add the spec's exact tokens as CSS custom properties:
  void `#0a0a0a`, chamber `#111111`, terminal `#0d0d0d`, text `#e8e4df` / muted `#6b6560`,
  accent sage `#6b7c6e`, gate border `#2d2020`. Reference tokens everywhere (no scattered hex).

## Design / craft notes

- Mobile-first, max content width 720px, reflections max-width 640px centered, line-height 1.8.
- 44px tap targets, visible focus states.
- NO chat bubbles, avatars, typing indicators. Reflection = a single contemplative letter block.
- "Lock this decision" must feel heavy/irreversible. Accepted gate = brief sage flash.
- `DemoNote` banner in chamber: "Prototype: all 9 weeks in one session. Production unlocks one week at a time."
- Reuse existing `src/app/components/ui` primitives (Textarea, Button) as styled building blocks where they fit the dark ritual aesthetic; otherwise custom.

## Done criteria (verification)

Manually walk the app via the preview surface and confirm every checklist item from the
spec's DONE CRITERIA section:

- "I'm considering leaving my job" → COMMITMENT GATE rejects (hypothetical).
- "I already resigned from my job last month" → accepted, locked forever, no edit UI anywhere.
- "Did I make a mistake?" / "Should I go back?" → RE-LITIGATION GATE rejects.
- "What if I had stayed?" → ANTI-COMPARE GATE rejects.
- Valid introspective question → Claude reflection generates (verify edge fn live), week advances.
- Week 9 complete → TERMINAL, **zero input elements in the DOM**.
- Export downloads a clean `.md` Living Record (decision + all 9 weeks + disclaimers).
- "Begin new arc" saves to Archive; archive tombstone is read-only.
- Disconnect network → reflection still appears via template fallback (demo safety).
- Overall: feels like a ritual chamber, not a chat app.
