# Living With It

A nine-week reflective instrument for people who already made an irreversible decision and cannot stop re-litigating it.

**Live demo:** https://living-with-it.vercel.app

## Run locally

```bash
npm install
npm run dev
```

## Built with

React, Vite, TypeScript, Tailwind CSS, Supabase, Groq, Resend, Vercel
